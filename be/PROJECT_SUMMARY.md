# 🎧 ATC Audio Agent

Real-time Air Traffic Control audio processing with AI-powered transcription and language understanding.

## Quick Start
```bash
git clone https://github.com/ai-berkeley-2025/atc-audio-agent.git
cd atc-audio-agent
make setup
# Edit .env with your Groq API key
make run
```
